The alt text field allows project owners to provide a short description of the image.

About

  • In the alt text field, you can write the alt text for the page image.

  • This feature is available for field-based or metadata description projects.

  • The character limit for the alt text field is 180 characters.

  • When you hit 125, the alt text field will turn yellow.

  • The visual indicator changes to orange when you hit 140 characters.

  • When you exceed 180, the visual indicator turns red.

  • The hard limit is at 255 characters.

Enabling

  1. Go to the Settings tab of your collection.

  2. Scroll down to Transcription Type and select "Enable Field Based Transcription".

  3. In the Input type dropdown for a field, select "alt text".

Exporting

Alt-text fields can be exported in spreadsheets on the Export tab of your collection.
